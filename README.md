## Описание проекта

Этот проект демонстрирует простую **свёрточную нейросеть (CNN)** на Go, которая обучается распознавать **рукописные цифры**.

### Что делает сеть

1. **Свёрточный слой (Conv2D)**

   * Применяет несколько фильтров к изображению, чтобы выявить **особенности** (границы, линии, штрихи).
   * На выходе получаются **карты признаков**, которые показывают, где на изображении встречаются эти элементы.

2. **Активация (ReLU)**

   * Пропускает только положительные значения, обнуляет отрицательные.
   * Помогает сети учиться сложным признакам.

3. **Пулинг (MaxPool2x2)**

   * Уменьшает размер карт признаков вдвое.
   * Сохраняет **самые важные признаки** (максимальные значения в каждом блоке).
   * Убирает лишние детали и делает сеть **устойчивой к смещению или шуму**.

4. **Flatten**

   * Превращает карты признаков в один длинный вектор, чтобы подать на полносвязный слой.

5. **Полносвязный слой (Dense)**

   * Суммирует признаки с весами и добавляет смещение (bias).
   * На выходе получаем **логиты** — «сырые оценки» принадлежности к каждому классу.

6. **Softmax**

   * Преобразует логиты в **вероятности** для каждого класса (сумма = 1).
   * Сеть «говорит», насколько уверена, что изображение принадлежит к конкретной цифре.

7. **Cross-Entropy Loss**

   * Сравнивает предсказание сети с правильной меткой и вычисляет **ошибку**, чтобы сеть могла корректировать веса.

### Что сеть ищет

* Она **ищет признаки цифр** в изображении:

  * линии, штрихи, пересечения
  * их расположение и направление
* На основе этих признаков сеть делает **предсказание класса** (0–9).
* Каждое изображение преобразуется через фильтры, пулинг и Dense слой, чтобы сеть могла понять, **какая цифра изображена**.

### Дополнительно

* Сеть выводит:

  * **Предсказанный класс**
  * **Вероятность предсказания**
  * **Совпадение с меткой** (для теста)
  * **Вероятности по всем классам** — можно видеть, на что сеть «смотрит» и как распределяет уверенность.
